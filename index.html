Qualtrics survey styles

Set of CSS overrides that can be applied to a qualtrics survey to enchance the default styles to match the Government Design System (GDS).

Only a subset of the available question types are supported, these are:

  • Information page - Used for introduction / information pages
  • Text input - For a single line or small amount of text
  • Text area - For answers over multiple lines or large amount of text
  • Checkboxes - Multiple choice / multiple selections
  • Radios - Multiple choice / single selection
  • Submit page - Used for submission confirmation pages

Additional styles

There are additional styles for error messages and a custom footer.
